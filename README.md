# Project

## Run the application

```shell
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 src/main.py
```

## Train the model

```shell
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 src/train_vit.py
```

## Eval the model

```shell
python -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
python3 src/eval_vit.py
```
